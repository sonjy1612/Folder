
# 휴가놈 Application Blueprint

## Overview

This application displays a full calendar for the year 2026. It is built using modern web technologies to be clean, readable, and visually appealing.

## Project Structure

*   `index.html`: The main HTML file.
*   `style.css`: The main stylesheet for the calendar.
*   `main.js`: The main JavaScript file, containing the logic for generating the calendar.
*   `blueprint.md`: This file, containing the project blueprint.

## Design and Features

### Visual Design

*   **Layout:** The calendar will display all 12 months of 2026 in a grid format.
*   **Typography:** A clean, modern font will be used for all text.
*   **Color Scheme:** A minimalist color scheme will be used, with weekends potentially highlighted for better readability.

### Features

*   **Full Year Calendar:** Displays all 12 months of 2026.
*   **Accurate Day Placement:** Each month correctly shows the day of the week for the 1st.
*   **Dynamic Generation:** The calendar is generated dynamically using JavaScript, making the code maintainable.

## Current Plan

*   Create the initial HTML structure for the calendar application.
*   Implement the calendar generation logic in `main.js`.
*   Style the calendar using modern CSS techniques to create a clean and readable layout.
